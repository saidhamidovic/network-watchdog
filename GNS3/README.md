# GNS3 Network Examinations & Labs

This directory contains a collection of network architecture and security examinations performed using GNS3, VMware, and KVM. Each project includes a detailed technical report in PDF format.

## 📁 Projects Overview

### 1. [Firewalls (Brandväggar)](./Hemtenta_Brandväggar_SN25_Said_Hamidovic.pdf)
- **Focus:** Linux firewall configuration using `iptables` and `nftables`.
- **Labs Included:** 
  - Configuration of Debian-based firewalls.
  - Security testing against Port Scanning, ICMP DoS/Flood, and SSH Brute Force using Kali Linux.
  - Version control of firewall rules using Git.

### 2. [Routing & Switching Basic](./Hemtentamen_Routing_Switching_Bas_Said_Hamidovic.pdf)
- **Focus:** Fundamental network topology and static routing.
- **Labs Included:** 
  - Building a network with 2 routers, 2 switches, and 4 end-users.
  - IP addressing schemes and static routing table configuration in GNS3.

### 3. [Server Administration](./Hemtentamen_Serveradministration_SN25_Said_Hamidovic.pdf)
- **Focus:** Multi-OS server management and network services.
- **Labs Included:** 
  - Package management comparison between Ubuntu Server (Linux) and FreeBSD (UNIX).
  - NFS (Network File System) configuration for cross-platform file sharing.
  - Integration with Windows 10 clients in a simulated GNS3 environment.

### 4. [Virtualization (Virtualisering)](./Hemtentamen_Virtualisering_SN25_Said_Hamidovic.pdf)
- **Focus:** Hypervisor technology and VM management.
- **Labs Included:** 
  - Comparison of VMware (Windows) and KVM (Linux) virtualization.
  - Deep dive into Hypervisor resource management (CPU/Memory allocation).
  - Implementation of isolated "sandbox" environments for secure testing.

---
*These reports represent practical examinations from the SN25 program at Folkuniversitetet Göteborg.*
